Como usar


El programa HUB ha sido pensado como un programa que permita agrupar las diferentes aplicaciones de resolucion de ejercicios de la asignatura Programadores del lenguaje 2. Pero en realidad puede usarse y es compatible con cualquier aplicación Java.

La venta principal "Apps" permite lanzar y configurar hasta 6 aplicaciones diferentes. El menú lateral izquierdo permite acceder a la confoguración y la ayuda, así como cerrar el programa.

A través del botón de configuración se permite modificar las características del programa. Actualmente sólo se puede modificar el idioma entre inglés y español.

Si se configura algún programa o imagen con una ruta inválida, o se cambia posteriormente tras haber configurado en el HUB, es posible que el HUB no sea capaz de iniciarse.

Si esto ocurriera, o si hubiera algún otro problema, se puede modificar manualmente el archivo "settings.conf" o restaurarlo a su versión inicial usando el archivo "reset_settings.bat", en la carpeta principal del HUB.
